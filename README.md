# 라이딩스타 라디오 시나리오

라디오 방송 회차별 시나리오와 큐시트를 웹페이지에서 확인하고 인쇄할 수 있도록 관리하는 정적 페이지입니다.

## 현재 메인 회차

- 방송일시: 2026년 8월 예정
- 녹음일시: 확인 필요
- 프로그램: 라이딩스타
- 진행자: 박정규, 강왕규
- 출연: 벨로혁신 부회장, 벨로혁신 총무
- 주제: 벨로혁신, 두 바퀴로 이어지는 사람들
- 예상 러닝타임: 60분
- 상태: 초안

## 회차 파일

- `index.html`: 벨로혁신 · 두 바퀴로 이어지는 사람들 웹페이지
- `kim-mikyung.html`: 김미경 · 자전거와 운동이야기 웹페이지
- `tangamja.html`: 김길중 · 탄감자와 자전거 챌린지 이야기 웹페이지
- `dubase.html`: 두바세 · 춘천 자전거 모임 이야기 웹페이지
- `episodes/2026-08-velo-hyeoksin.md`: 벨로혁신, 두 바퀴로 이어지는 사람들
- `episodes/2026-06-15-tangamja.md`: 탄감자와 자전거 챌린지 이야기
- `episodes/2026-06-05-riding-star.md`: 자전거와 운동이야기
- `episodes/dubase.md`: 두바세를 만나다

## 화면 사용

- 좌측 회차 관리 메뉴에서 게스트 이름을 클릭하면 오른쪽 화면이 해당 회차로 전환됩니다.
- 오른쪽 상단 탭에서 선택된 회차의 큐시트, 전체 대본, 질문 카드, 진행표, 확인 항목을 따로 확인합니다.

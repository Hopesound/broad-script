- 방송일시: 2026년 6월 15일
- 녹음일시: 확인 필요
- 프로그램: 라이딩스타
- 진행자: 박정규, 강왕규
- 출연: 김길중
- 주제: 탄감자와 자전거 챌린지 이야기
- 예상 러닝타임: 60분
- 상태: 초안

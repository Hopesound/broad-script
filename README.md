라디오 방송용 시나리오를 회차별로 저장하고 관리하는 공간입니다.

## 회차 목록

| 방송일 | 제목 | 상태 | 파일 |
| --- | --- | --- | --- |

## 관리 방식

- 새 회차는 `episodes/YYYY-MM-DD-riding-star.md` 형식으로 저장합니다.
- 방송일, 진행자, 게스트, 주제, 상태를 파일 상단에 기록합니다.
- 실제 방송 전에는 `확인 필요` 항목을 점검하고 상태를 갱신합니다.
